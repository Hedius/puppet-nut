Puppet Class: nut

Inherits:
::nut::params
Defined in:
manifests/init.pp

Overview

Installs Network UPS Tools (NUT).

Examples:

Declaring the class

include ::nut

Listening on local subnet

class { '::nut':
  listen => [
    {
      'address' => $::ipaddress_eth0,
    },
    {
      'address' => $::ipaddress6_eth0,
    },
  ],
}
